Во втором законе непрерывности в гештальт-психологии, говорится, что в процессе восприятия люди обычно склонны понимать объект так, как он сначала воспринимается, позволяя прямым линиям быть прямыми, а кривым - кривыми. <br />
В дизайне интерфейса выравнивание элементов соответствует восприятию пользователей и делает информацию более доступной для понимания.

## Выравнивание табличных данных <a name="text-table"></a>

Используйте следующие правила:

1. Числовые данные выравниваются по правой стороне.
2. Текстовые данные выравниваются по левой стороне.
3. Заголовки выравниваются в соответствии с данными.
4. Выравнивание по центру не используется.
